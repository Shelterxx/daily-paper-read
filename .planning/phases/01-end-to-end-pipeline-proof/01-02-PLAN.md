**2. `src/search/arxiv_source.py`** -- ArXiv adapter using the `arxiv` library with `feedparser` as fallback:

Key implementation details:
- Use `arxiv` library (version 2.4.0+) as primary search interface
- For each keyword in the SearchQuery.keywords list, run a separate search
- Use `asyncio.gather()` to run multiple keyword searches in parallel (respecting arXiv rate limit of 1 req/s)
- Map arXiv results to the Paper model with `source="arxiv"`, `paper_id` = arXiv ID, `pdf_url` = `https://arxiv.org/pdf/{id}`
- Filter results by `published_date` within the `SearchQuery.timeframe_hours`
- Set `source_url` = `https://arxiv.org/abs/{id}`
- Use `tenacity` for retry with exponential backoff on HTTP errors (retry on 429, 500, 502, 503; max 3 attempts; base delay 2s with jitter)
- If `arxiv` library raises unexpected errors, fall back to `feedparser` with raw HTTP via `httpx`
- Log each search query and result count via stdlib `logging`

```python
from src.search.models import Paper

def deduplicate_papers(papers: list[Paper]) -> list[Paper]:
    """Remove duplicate papers by dedup_key, keeping first occurrence."""
    seen = set()
    result = []
    for paper in papers:
        if paper.dedup_key not in seen:
            seen.add(paper.dedup_key)
            result.append(paper)
    return result
```

**2. `src/fetch/pdf_fetcher.py`** -- Async PDF download with graceful fallback:

```python
import logging
import httpx
from typing import Optional
from tenacity import retry, stop_after_attempt, wait_exponential, retry_if_exception_type

logger = logging.getLogger(__name__)

PDF_TIMEOUT_SECONDS = 30
MAX_PDF_SIZE_BYTES = 50 * 1024 * 1024  # 50MB
PDF_MAGIC_BYTES = b"%PDF"

@retry(
    stop=stop_after_attempt(2),
    wait=wait_exponential(multiplier=1, min=1, max=10),
    retry=retry_if_exception_type((httpx.TimeoutException, httpx.HTTPStatusError)),
)
async def fetch_pdf(url: str, client: httpx.AsyncClient) -> Optional[bytes]:
    """Download PDF from URL. Returns bytes if successful, None on failure.

    Validates response is actually a PDF by checking magic bytes.
    Respects size limits to avoid memory issues in GitHub Actions.
    """
    try:
        response = await client.get(url, timeout=PDF_TIMEOUT_SECONDS, follow_redirects=True)
        response.raise_for_status()

        content = response.content
        if len(content) > MAX_PDF_SIZE_BYTES:
            logger.warning(f"PDF too large ({len(content)} bytes): {url}")
            return None

        if not content[:4] == PDF_MAGIC_BYTES:
            logger.warning(f"Response is not a valid PDF: {url}")
            return None

        logger.info(f"Downloaded PDF ({len(content)} bytes): {url}")
        return content
    except httpx.TimeoutException:
        logger.warning(f"PDF download timeout: {url}")
        return None
    except httpx.HTTPStatusError as e:
        logger.warning(f"PDF download HTTP {e.response.status_code}: {url}")
        return None
    except Exception as e:
        logger.warning(f"PDF download failed: {url} - {e}")
        return None


async def fetch_and_enrich_paper(paper, client: httpx.AsyncClient) -> None:
    """Attempt to fetch PDF for a paper and enrich it with full_text.

    Mutates paper.full_text and paper.text_source in place.
    If PDF fetch fails, paper.text_source remains 'abstract'.
    """
    if not paper.pdf_url:
        paper.text_source = "abstract"
        return

    pdf_bytes = await fetch_pdf(paper.pdf_url, client)
    if pdf_bytes:
        from src.fetch.text_extractor import extract_text_from_pdf
        text = extract_text_from_pdf(pdf_bytes)
        if text and len(text.strip()) > 100:  # Minimum meaningful text
            paper.full_text = text
            paper.text_source = "full_text"
            return

    paper.text_source = "abstract"
    logger.info(f"Using abstract fallback for: {paper.title[:50]}")
```

**3. `src/fetch/text_extractor.py`** -- PyMuPDF wrapper:

```python
import io
import logging
from typing import Optional

logger = logging.getLogger(__name__)

def extract_text_from_pdf(pdf_bytes: bytes) -> Optional[str]:
    """Extract text content from PDF bytes using PyMuPDF.

    Returns extracted text string, or None if extraction fails.
    Handles multi-page PDFs by concatenating all pages.
    """
    try:
        import fitz  # PyMuPDF
    except ImportError:
        logger.warning("PyMuPDF not installed, cannot extract PDF text")
        return None

    try:
        doc = fitz.open(stream=io.BytesIO(pdf_bytes), filetype="pdf")
        text_parts = []
        for page_num in range(len(doc)):
            page = doc[page_num]
            text = page.get_text("text")
            if text.strip():
                text_parts.append(text.strip())

        doc.close()
        full_text = "\n\n".join(text_parts)

        if not full_text.strip():
            logger.warning("PDF text extraction produced empty output")
            return None

        return full_text
    except Exception as e:
        logger.warning(f"PDF text extraction failed: {e}")
        return None
```

Key design choices per PITFALLS.md:
- PDF magic bytes check (`%PDF`) prevents processing non-PDF responses
- 50MB size limit prevents memory issues in GitHub Actions
- 30-second timeout per download
- Minimum 100 chars of extracted text to be considered valid (avoids blank/title-only extraction)
- Graceful fallback: if anything fails, paper.text_source = "abstract" and the pipeline continues
