    Key implementation details:

    **write_and_push** (main entry):
    1. Check `self.config.enabled` -- if False, log "Obsidian disabled, skipping" and return.
    2. Check `self.config.vault_repo_url` is non-empty -- if empty, log warning and return.
    3. Read PAT from `os.environ.get(self.config.vault_pat_env, "")`.
    4. Get today's date string in UTC: `date_str = datetime.now(timezone.utc).strftime('%Y-%m-%d')`.
    5. Build topic peers map via `_get_topic_peers`.
    6. Create a temporary directory: `with tempfile.TemporaryDirectory() as tmpdir:`
    7. Create `papers/` and `daily/` subdirectories in tmpdir.
    8. For each AnalyzedPaper:
       - Generate card via `_generate_paper_card(ap, topic_peers[ap.topic_name])`.
       - Write to `tmpdir/papers/{sanitized_filename}.md`.
       - Track stats: cards_written, errors.
    9. Generate daily summary via `_generate_daily_summary`.
    10. Write to `tmpdir/daily/{date_str}-daily-summary.md`.
    11. Call `_git_clone_push(tmpdir, date_str, cards_written)`.
    12. Return stats dict.
    Wrap the entire method in try/except -- errors are logged, not raised.

    **_sanitize_filename**:
    - If paper has DOI: use `paper.doi.replace('/', '-').replace('\\', '-')`
    - Else: use `paper.dedup_key.replace(':', '-').replace('/', '-')`
    - Remove characters that are invalid in filenames: `< > : " | ? *`
    - Result is the base filename without `.md` extension.

    **_generate_paper_card** (OBS-01, OBS-03):
    - Build YAML frontmatter:
      ```yaml
      ---
      title: "{paper.title}"
      authors: [{comma-separated authors}]
      doi: "{paper.doi or ''}"
      score: {analysis.relevance_score}
      date: {date_str}
      topics: ["{ap.topic_name}"]
      keywords: [{comma-separated extracted_keywords}]
      source: {paper.source}
      ---
      ```
    - Build body sections. Use `## Section Name` headers. Content comes from analysis fields:
      - `## Summary`: `analysis.summary` or "(No summary available)"
      - `## Key Contributions`: bulleted list from `analysis.key_contributions` or skip section
      - `## Methodology`: `analysis.methodology_evaluation` or skip section
      - `## Limitations`: bulleted list from `analysis.limitations` or skip section
      - `## Future Directions`: bulleted list from `analysis.future_directions` or skip section
    - `## Related Papers` section (OBS-03 backlinks):
      - For each peer filename in `topic_peers` (excluding self):
        Add `[[{peer_filename}]]` on its own line.
      - Add `[[{ap.topic_name}]]` as a topic tag link.
    - Use the `language` setting for section headers? No -- per CONTEXT.md, use English section headers but content stays in the analysis language (Chinese if configured). The template structure is fixed in the CONTEXT.md.

    **_generate_daily_summary** (OBS-02):
    - Header: `# Daily Literature Digest - {date_str}`
    - Statistics block:
      ```
      ## Statistics
      - Total new papers: {total}
      - HIGH: {high_count}, MEDIUM: {medium_count}, LOW: {low_count}
      - Archived to Zotero: {zotero_archived} (note: this info may not be available yet, so omit if not passed)
      ```
    - For each topic, list papers sorted by score (descending):
      ```
      ## {topic_name}
      | Score | Title | Link |
      |-------|-------|------|
      | {score} | [[{card_filename}\|{title}]] | [source]({source_url}) |
      ```
      Use wiki-link syntax `[[filename|display_text]]` for each paper.
    - Add a "## All Papers" section at the end with simple bulleted list of all wiki-links.

    **_get_topic_peers** (OBS-03):
    - Group papers by `topic_name`.
    - For each group, produce a list of sanitized filenames.
    - Return dict: `topic_name -> [filename1, filename2, ...]` where filenames do NOT include `.md`.

    **_git_clone_push** (OBS-04):
    - Construct authenticated URL: `repo_url.replace('https://', f'https://{pat}@')`
    - Clone the vault repo into a new temp directory:
      ```python
      vault_dir = Path(tempfile.mkdtemp())
      subprocess.run(['git', 'clone', auth_url, str(vault_dir)], capture_output=True, check=True, timeout=60)
      ```
    - Copy generated files from local_dir to vault_dir:
      - Copy `papers/*.md` to `vault_dir/papers/` (create if needed)
      - Copy `daily/*.md` to `vault_dir/daily/` (create if needed)
      - For existing paper cards (same filename), overwrite the content (per CONTEXT.md dedup rule: update, don't create new).
    - Commit:
      ```python
      subprocess.run(['git', 'add', 'papers/', 'daily/'], cwd=str(vault_dir), capture_output=True)
      subprocess.run(['git', 'commit', '-m', f'docs(daily): {date_str} -- {paper_count} papers archived'],
                     cwd=str(vault_dir), capture_output=True)
      ```
    - Push to main:
      ```python
      subprocess.run(['git', 'push', 'origin', 'main'], cwd=str(vault_dir), capture_output=True, timeout=60)
      ```
    - Conflict handling: if push fails, try `git pull --rebase origin main` then push again once. If still fails, log error and return False.
    - Clean up: delete the cloned vault_dir temp directory.
    - Return True on success, False on failure.
