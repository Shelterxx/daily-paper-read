Create `src/search/doi_resolver.py` with:

1. **`async def enrich_paper_from_doi(paper: Paper, client: httpx.AsyncClient) -> None`**:
   - If `paper.doi` is None, return immediately (nothing to resolve)
   - If paper already has title + abstract + authors (all non-empty), return (nothing to enrich)
   - Make GET request to `https://doi.org/{paper.doi}` with headers:
     ```python
     {"Accept": "application/citeproc+json"}
     ```
   - Use the shared `client` parameter (httpx.AsyncClient)
   - Timeout: 15 seconds (metadata is small)
   - `follow_redirects=True` (DOI.org redirects)
   - tenacity retry: 2 attempts, exponential backoff on timeout/5xx
   - On failure: log warning, return without modifying paper (graceful)
   - Parse response as JSON. citeproc+json format provides:
     - `title`: str
     - `author`: list of `{"given": "First", "family": "Last"}`
     - `abstract`: str (may not always be present)
     - `DOI`: str
     - `published-print` or `published-online`: `{"date-parts": [[YYYY, MM, DD]]}`
     - `URL`: str
   - Enrich paper fields only where they are currently missing/empty:
     - If `not paper.title`: set from `response["title"]`
     - If `not paper.abstract`: set from `response.get("abstract")`
     - If `not paper.authors`: set from `response.get("author", [])` -> `["{given} {family}" for each]`
     - If `not paper.published_date`: try to parse from `response.get("published-print", {}).get("date-parts", [[None]])[0]` -> datetime
     - If `not paper.source_url`: set from `response.get("URL")`
   - Log: `"Enriched paper {doi} with {n} fields from DOI content negotiation"`
   - This function mutates the Paper object in place (same pattern as fetch_and_enrich_paper)

2. **`async def enrich_papers_batch(papers: list[Paper], client: httpx.AsyncClient) -> None`**:
   - Enrich all papers that need DOI resolution
   - Use `asyncio.Semaphore(5)` to limit concurrent requests
   - Use `asyncio.gather(*[...], return_exceptions=True)` for parallel execution
   - Each individual failure is logged but does not affect others
   - Log total: `"Enriched {n}/{total} papers from DOI content negotiation"`

1. **Create `src/fetch/multi_channel_fetcher.py`:**

   **`async def resolve_unpaywall_pdf_url(doi: str, client: httpx.AsyncClient) -> Optional[str]`**:
   - Read `UNPAYWALL_EMAIL` from `os.environ`. If not set, return None (Unpaywall requires email).
   - GET `https://api.unpaywall.org/v2/{doi}?email={email}`
   - Timeout: 15s, `follow_redirects=True`
   - Parse response JSON, look for `best_oa_location` -> `url_for_pdf` (or `url` if `url_for_pdf` is None)
   - If found, return the PDF URL. If not, return None.
   - On any error: log warning, return None.

   **`async def resolve_pmc_pdf_url(pmid: str, client: httpx.AsyncClient) -> Optional[str]`**:
   - This is a secondary channel for papers with PMC IDs
   - GET `https://www.ncbi.nlm.nih.gov/pmc/utils/oa/oa.fcgi?id={pmid}`
   - Actually, simpler approach: if we have a PMC ID, the PDF URL pattern is:
     `https://www.ncbi.nlm.nih.gov/pmc/articles/{pmid}/pdf/`
   - Just return the constructed URL. The actual download will validate via fetch_pdf.
   - If `pmid` doesn't start with "PMC", prepend it.

   **`async def fetch_pdf_multi_channel(paper: Paper, client: httpx.AsyncClient) -> None`**:
   - This replaces `fetch_and_enrich_paper` for multi-source papers.
   - Implements a priority chain of PDF URL resolution and download:
     ```
     Channel 1: paper.pdf_url (already set by search adapter from S2 openAccessPdf or OpenAlex oa_url)
       -> if paper.pdf_url is set, try fetch_pdf(paper.pdf_url, client)
     
     Channel 2: Unpaywall (if paper.doi is set)
       -> resolve_unpaywall_pdf_url(paper.doi, client) -> if URL found, try fetch_pdf(url, client)
     
     Channel 3: PMC (if paper has a pmc_id in source metadata -- stored as paper_id if source is pubmed/pmc)
       -> resolve_pmc_pdf_url(pmc_id, client) -> if URL found, try fetch_pdf(url, client)
   ```
   - After any successful download (pdf_bytes is not None):
     a. Use `extract_text_from_pdf(pdf_bytes)` to get text
     b. If text is not None and `len(text.strip()) > 100`: set `paper.full_text = text`, `paper.text_source = "full_text"`, return
   - If all channels fail: set `paper.text_source = "abstract"`, log info message
   - Log each channel attempt: `"Trying {channel_name} for {paper.title[:40]}..."` and `"Channel {channel_name} {'succeeded' if ok else 'failed'}"`

   **Important**: Import `fetch_pdf` from `src.fetch.pdf_fetcher` for the actual download. Do NOT duplicate the download+validation logic. This module only handles URL resolution; `fetch_pdf` handles download + magic bytes + size validation.
